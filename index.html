Tables Providing a Quick HTML Guide

The Basics

Tags or Attributes Description
<!DOCTYPE html> Defines document type as HTML
<html> Defines the root of an HTML document
<head> Contains metadata for the document

All About Tables

Tags or Attributes Description
<table> Allows table creation to begin
<tr> Creates a row of cells
<td> Creates a data cell
<thead> Creates space for table header
<th> Creates a header cell
<tbody> Creates space for table body
<tfoot> Creates a table footer
colspan Attribute allowing cell data to span columns
rowspan Attribute allowing cell data to span rows
